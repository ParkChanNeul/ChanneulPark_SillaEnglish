## 1. Brand Core

### One-line Definition

**외운 영어가 아니라, 대본 없이 말하는 영어.**

### Positioning

신라어학원은 초등 전환기에 말하기를 잃지 않게 만드는 성장관리 어학원이다. 대형학원의 객관적인 리딩 지표와 동네 학원의 밀착 케어 사이에서, 아이가 자기 생각을 영어로 구조화하고 발표하게 만드는 것을 핵심 가치로 둔다.

### First Target

- 영유 또는 영어노출 경험이 있는 초1-3 학부모
- 아이가 영어를 알아듣지만 말로 꺼내는 힘이 약하다고 느끼는 가정
- 대형학원의 성적표는 받지만 아이의 말하기 변화는 보이지 않는다고 느끼는 가정

### First Conversion

**무료 말하기 진단 신청**

진단은 아이의 현재 수준을 보여주는 첫 경험이다. 홈페이지, 블로그, 브로셔, 설명회 자료는 모두 이 신청으로 이어져야 한다.

## 2. Message Hierarchy

### Primary Promise

**대본 없이 자기 생각을 영어로 말하는 아이.**

### Supporting Proof

1. 말하기 - 구조 - 정확성 순서의 수업 설계
2. 마인드맵 - 스크립트 - 발표 - 피드백 발표 시스템

### What Not To Lead With

- "둔촌동 최고 프리미엄"
- "대치동식 관리"
- "원장 경력"
- "SR 테스트"

이 요소들은 신뢰를 만든다. 하지만 첫 문장은 아이의 변화여야 한다.

## 3. Visual Thesis

### Direction

**Minimal Premium with Active Clarity**

고급스럽지만 조용하고, 아이의 말하기 수업이 가진 움직임을 절제된 방식으로 담는다. 로고는 특정 소재를 맞히는 문제가 아니다. 신라어학원이 무엇을 변화시키는지 조형적으로 설득하는 문제다.

### Core Interpretation

**구조의 문장**

아이의 머릿속에 흩어진 영어가 구조화되고, 자기 생각으로 정리되어, 입 밖으로 나오는 과정을 시각화한다. 이 해석은 특정 심볼을 제한하지 않는다. 어떤 소재나 조형을 쓰더라도 이 과정과 연결되어야 한다.

### Mood

- 학부모에게: 믿음, 절제, 관리감
- 아이에게: 자신감, 발표감, 너무 딱딱하지 않은 에너지
- 시장에서: 흔한 영어학원처럼 보이지 않는 선명함

## 4. Composite Logo System

로고 시스템은 `심볼 + 신라어학원 + SILLA English` 복합 로고를 전제로 운영한다. 특정 심볼 소재를 정답으로 두지는 않는다. 디자이너는 최소 2개 이상의 다른 방향을 제안하고, 각 방향의 장단점을 비교해야 한다.

### Fixed Elements

- Korean name: `신라어학원`
- English detail: `SILLA English`
- Core message: `외운 영어가 아니라, 대본 없이 말하는 영어.`
- Brand frame: `구조의 문장`
- Logo format: `심볼 + 신라어학원 + SILLA English`

### Open Elements

- 심볼의 소재와 은유
- 심볼의 강도, 위치, 비례
- 한글/영문 비중
- 워드마크의 성격
- 유틸리티 마크 필요 여부
- 간판형, 브로셔형, 모바일형 분화 여부

### Expected Lockups

- Korean primary lockup
- Korean + English lockup
- Compact lockup for mobile
- One-color black and white versions
- Application examples for sign, brochure, website header, report cover

### Current Logo Directions

현재 로고 탐색은 모두 `심볼 + 신라어학원 + SILLA English` 복합 로고를 전제로 한다. 두 시안의 차이는 소재가 아니라 브랜드 인상으로 나눈다. 1안은 신뢰형, 2안은 감도형으로 충분히 다르게 보여야 한다.

## 6. Color

### Palette Strategy

컬러는 로고 시안의 정답이 아니라 적용 환경이다. 1안과 2안 모두 먼저 흑백과 차콜 단색에서 성립해야 한다. 컬러는 그 다음에 신뢰감, 프리미엄감, 리포트 확장성을 보강하는 용도로만 쓴다.

이전 팔레트의 딥 틸과 코랄은 말하기의 활동감을 설명하기에는 좋지만, 현재 확정한 복합 로고 방향에서는 초등 액티비티 학원이나 캠페인성 브랜드처럼 보일 위험이 있다. 로고와 핵심 브랜드 시스템에서는 제외하고, 필요할 때 이벤트 배너나 일회성 콘텐츠에서만 별도 검토한다.

| Token | Hex | Usage |
|-------|-----|-------|
| Warm Ivory | `#F7F2E8` | 배경, 브로셔 종이감, 설명회 표지 |
| Paper White | `#FFFDF8` | 카드, 리포트, 폼 |
| Ink Charcoal | `#111418` | 1차 로고, 제목, 고급 단색 적용 |
| Academic Navy | `#142236` | 1안의 기관 신뢰감, CTA, 깊은 배경 |
| Soft Graphite | `#2B2F34` | 본문, 보조 텍스트, 2안의 조용한 적용 |
| Quiet Silver | `#BFC4C9` | 구분선, Silver 단계, 리포트 UI |
| Silla Gold | `#B89A5E` | 성취, Gold 단계, 제한된 포인트 |
| Warm Gray | `#8A8176` | 보조 캡션, 종이 질감 위의 약한 정보 |

### Logo Color Application

| Direction | Primary Use | Accent Use | Avoid |
|-----------|-------------|------------|-------|
| 1안 Academic Structure Mark | Ink Charcoal or Academic Navy 단색 | Quiet Silver, Silla Gold를 리포트/레벨 배지에만 절제 | 금색 로고 면 채움, 밝은 포인트 컬러 |
| 2안 Boutique Voice Mark | Ink Charcoal 단색, Warm Ivory 반전 | Warm Gray, Silla Gold를 인쇄 후가공 또는 작은 디테일로만 사용 | 네이비 면적 과다, 캠페인성 액센트 |

### Level Colors

- White: `#F8F6F0` with Ink Charcoal text
- Silver: `#C8CDD2` with Ink Charcoal text
- Gold: `#B89A5E` with Ink Charcoal or Paper White text

레벨 컬러는 로고 컬러가 아니라 학습 단계와 리포트 UI를 위한 색이다. White / Silver / Gold를 브랜드 전면 컬러처럼 쓰면 프리미엄보다 단계표 느낌이 강해진다.

### Usage Ratio

- Warm Ivory/Paper White: 60%
- Ink Charcoal/Academic Navy/Soft Graphite: 32%
- Quiet Silver/Warm Gray: 5%
- Silla Gold: 3%

색상 비율은 로고 확정 전 임시 기준이다. 로고 자체는 1색으로 검증하고, 컬러 면적은 브로셔, 웹, 리포트 같은 적용물에서만 조절한다.

## 7. Typography

로고는 완성형 폰트 입력이 아니라 커스텀 레터링을 목표로 한다. 한글 워드마크는 기하학적 산세리프를 기반으로 하되 획 끝과 자간을 조정해 학원 로고처럼 보이는 저가 느낌을 피한다.

## 8. Layout

### Approach

Grid-disciplined, editorial-light.

학부모가 빠르게 신뢰할 수 있도록 정보 구조는 명확하게 둔다. 하지만 입시 학원 전단처럼 빽빽하게 채우지 않는다. 여백은 프리미엄을 만들고, 표와 리포트 UI는 관리감을 만든다.

### Grid

- Desktop: 12 columns, max width 1180px
- Tablet: 8 columns
- Mobile: 4 columns
- Section vertical padding: 80px desktop, 48px mobile

### Radius

- Logo/wordmark: no rounded container unless the selected logo direction requires a formal seal system
- Cards: 8px max
- Buttons: 6px
- Badges: 999px only for level chips

## 9. Graphic Language

### Core Motifs

- Thin rule lines: 문장 구조와 리포트 느낌
- Bracket marks: 진단, 피드백, 교정의 느낌
- Level chips: White / Silver / Gold
- Mindmap lines: 발표 시스템 설명에서만 제한적으로 사용

### Evaluation

그래픽 요소는 금지 목록으로 판단하지 않는다. 아래 질문으로 판단한다.

- 브랜드가 더 신뢰감 있게 보이는가?
- 아이의 말하기 성장과 연결되는가?
- 리포트, 브로셔, 블로그까지 확장 가능한가?
- 설명을 듣기 전에도 조형적으로 완성도가 있는가?

## 11. Component Rules

### Buttons

- Primary CTA: Academic Navy or Ink Charcoal background, Paper White text
- Secondary CTA: transparent, Ink Charcoal border
- Diagnostic CTA label: `무료 말하기 진단 신청`

### Level Badges

- White/Silver/Gold는 작은 칩이나 리포트 헤더에서만 사용
- Gold를 남발하지 않는다. 프리미엄은 금색 면적이 아니라 비례와 여백에서 나온다.

### Reports

- 숫자와 그래프는 표준화된 리포트 UI로 보인다.
- "미국 학년 기준"은 부모가 이해하기 쉽게 `GE 3.2`처럼 숫자와 함께 해석 문장을 붙인다.
