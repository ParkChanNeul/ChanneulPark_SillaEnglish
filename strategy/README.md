# Strategy

신라어학원의 브랜드, 로고, 오픈 마케팅 판단을 관리한다.

## Files

| File | Role |
|------|------|
| `brand_design_system.md` | 컬러, 타이포, 로고, 그래픽, 사진, 컴포넌트 규칙 |
| `logo_design_brief.md` | 외부 로고/BX 디자이너에게 전달할 제작 브리프 |
| `open_marketing_plan.md` | 무료 말하기 진단 중심 오픈 전 퍼널 |
| `copy_system.md` | 홈페이지, 브로셔, 블로그, 설명회 카피 |

## Source of Truth

브랜드 디자인은 `brand_design_system.md`를 먼저 따른다. 마케팅 실행은 `open_marketing_plan.md`를 따른다. 문구는 `copy_system.md`에서 가져온다.
