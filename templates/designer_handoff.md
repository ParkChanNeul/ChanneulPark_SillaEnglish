# Designer Handoff - 신라어학원

## Read First

1. `strategy/brand_design_system.md`
2. `strategy/logo_design_brief.md`

## Fixed Inputs

- Korean name: `신라어학원`
- English detail: `SILLA English`
- Core message: `외운 영어가 아니라, 대본 없이 말하는 영어.`
- Brand frame: `구조의 문장`
- First target: 영유 또는 영어노출 경험이 있는 초1-3 학부모
- First conversion: 무료 말하기 진단 신청

## Creative Direction

두 시안 모두 `심볼 + 신라어학원 + SILLA English` 복합 로고로 제안한다. 심볼 소재는 제한하지 않는다. 어떤 소재, 은유, 글자 변형, 추상 조형도 제안할 수 있다.

단, 각 시안은 아래 문장과 연결되어야 한다.

> 아이가 머릿속에 흩어진 영어를 구조화하고, 자기 생각을 문장으로 만들어, 입 밖으로 끝까지 말하는 과정.

## Required Directions

| Direction | Role | Requirement |
|-----------|------|-------------|
| 1안 신뢰형 | 학부모 신뢰, 관리감, 오프라인 간판/상담/설명회 적합성 | 안정적인 복합 로고. 심볼은 구조화, 기준점, 성장감을 설명할 수 있어야 한다. |
| 2안 감도형 | 프리미엄 차별성, 기억성, 디지털/브랜드 확장성 | 더 과감한 복합 로고. 심볼은 글자 변형이나 추상 구조를 자유롭게 쓸 수 있다. |

## Required Designer Output

- Minimum 2 distinct composite logo directions
- Symbol treatment for each direction
- Korean wordmark treatment for each direction
- `SILLA English` treatment for each direction
- Horizontal lockup
- Stacked or cover lockup
- One-color black version
- One-color white version
- Small-size test
- Application examples: signage, brochure cover, website header, parent report cover

## Required Comparison Table

Each direction must include:

| Field | Requirement |
|-------|-------------|
| Intent | What brand impression this direction creates |
| Strengths | Where this direction is strongest |
| Risks | Where it may misread or weaken |
| Best Use Cases | Which surfaces it fits best |
| Refinement Direction | What to improve in the next round |

## Evaluation Criteria

- Premium trust: 월 30-38만 원 가격대가 시각적으로 납득되는가
- Speaking growth: 말하기, 발표, 자기표현의 변화가 느껴지는가
- Structure: 생각이 정리되고 문장으로 완성되는 느낌이 있는가
- Distinctiveness: 지역 영어학원이나 프랜차이즈 템플릿처럼 보이지 않는가
- Scalability: 간판, 브로셔, 리포트, 블로그, 모바일에 확장되는가
- Small-size clarity: 작은 크기에서도 읽히거나 인지되는가
- One-color strength: 색 없이도 조형이 버티는가
- Parent clarity: 학부모가 설명을 들었을 때 납득하기 쉬운가

## Review Question

최종 선택은 취향이 아니라 이 질문으로 판단한다.

> 이 로고는 신라어학원이 아이의 영어를 구조화하고 말하게 만드는 곳이라는 믿음을 만드는가?
