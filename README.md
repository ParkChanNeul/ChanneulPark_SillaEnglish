# Shilla English Brand System

> `신라어학원` 브랜드 디자인, 로고, 오픈 마케팅 산출물을 관리하는 프로젝트 폴더.

## Core Position

**외운 영어가 아니라, 대본 없이 말하는 영어.**

신라어학원은 초등 전환기에 말하기를 잃지 않게 만드는 성장관리 어학원이다. 첫 타겟은 영유 또는 영어노출 경험이 있는 초1-3 학부모이며, 첫 전환 장치는 무료 말하기 진단이다.

## Rules

- 시안 확정 전 시각 에셋은 보관하지 않는다. 초안 SVG나 프리뷰가 남아 있으면 디자이너 탐색을 좁힐 수 있다.
- 로고 브리프는 특정 심볼을 제한하지 않는다.
- 모든 시안은 `구조의 문장`이라는 해석 프레임으로 장단점을 설명해야 한다.
- `SR 테스트`, 월말 성적표, 데일리 평가는 메인 메시지가 아니라 신뢰 증명 장치다.
- 모든 산출물은 무료 말하기 진단 신청으로 이어져야 한다.
